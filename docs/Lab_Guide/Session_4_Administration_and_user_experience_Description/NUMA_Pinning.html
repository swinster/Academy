Use section #3 of the lab sheet.

Go to https://docs.pexip.com --> Installation --> VMware Installation --> VMware NUMA Affinity (or just search NUMA pinning in the docs, you should be fluent at this by now).

Follow the instructions but be sure to adjust for the number of actual physical cores for your allocated NUMA node on your server. See Lab sheet section #3 – if no NUMA node is specified, you have the whole server for your group. If you require further clarification, ask your trainer.

As your Management Node is on the same server (perhaps even a single NUMA node), ensure you take account of its requirements too.

Make a CPU budget of how you plan to deploy. Consult your trainer if you would like some confirmation that you are on the right path (HINT: our docs say that you should only configure this with one node per socket, however, in the lab we will take some artistic licence as there will be multiple nodes per socket, so the goal here is to see if you can get the procedure correct).
