Use section #5 of the lab sheet.

Refer to https://docs.pexip.com/admin/about_h323_sip.htm and https://docs.pexip.com/admin/integrate_pstn.htm. 

Use the Pexip Infinity general Virtual Reception (VR) setup in session 2. This should have an alias of lobby@<your Pexip Infinity domain> (hopefully you know where to go by now). You will also want to handle a PSTN number as a separate alias in the VR. The PSTN number will depend on your trainer and location. Your trainer should be able to provide you with a PSTN number that will route to your VR (via Twilio VoIP).

You can then try to dial into a VMR using a phone.

The destination VMR must have one alias that is purely numeric so that users can enter it as touch-tones (DTMF) via the phone keypad. This will allow you to dial into the VR, then tap the numeric alias of the VMR to be transferred. This information would normally be distributed in a calendar invite, such as through using the Pexip Scheduling option.
