To support your OTJ Exchange integration, below are some useful PowerShell command snippets you can use:

Upgrade of packages for PowerShell
https://docs.microsoft.com/en-us/powershell/scripting/gallery/installing-psget?view=powershell-7


Updating PowerShell TLS Version 32 bit version:

Set-ItemProperty -Path 'HKLM:\SOFTWARE\Wow6432Node\Microsoft\.NetFramework\v4.0.30319' -Name 'SchUseStrongCrypto' -Value '1' -Type DWord


Updating PowerShell TLS Version 64 bit version:

Set-ItemProperty -Path 'HKLM:\SOFTWARE\Microsoft\.NetFramework\v4.0.30319' -Name 'SchUseStrongCrypto' -Value '1' -Type DWord


Checking PowerShell TLS version:

[Net.ServicePointManager]::SecurityProtocol
